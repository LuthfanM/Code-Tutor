# Catatan Belajar Golang

Go terasa sederhana karena syntax-nya kecil dan tooling-nya rapi.

## Mulai Dari CLI Kecil

Sebelum membuat API, coba buat program CLI kecil untuk memahami struct, function, dan error handling.

## Error Handling

Go membuat error handling eksplisit.

```go
if err != nil {
  return err
}
```

## Setelah Dasar

Lanjutkan ke HTTP server, database, dan testing.
