# Kenapa Belajar JavaScript Dulu

JavaScript memberi feedback cepat karena bisa langsung dijalankan di browser.

## Feedback Cepat

Saat belajar, feedback cepat membantu kamu memahami hubungan antara code dan hasil.

## Ekosistem Luas

JavaScript dipakai untuk frontend, backend, tooling, dan scripting kecil.

## Tetap Perlu Dasar

Walaupun mudah dimulai, tetap pelajari variable, function, array, object, dan async secara pelan-pelan.
