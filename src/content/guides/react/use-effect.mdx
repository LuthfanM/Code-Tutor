# React useEffect

`useEffect` dipakai untuk sinkronisasi, bukan untuk semua logic.

## Contoh Fetch

```tsx
useEffect(() => {
  async function loadData() {
    const response = await fetch("/api/items")
    setItems(await response.json())
  }

  loadData()
}, [])
```

## Dependency

Dependency menentukan kapan effect dijalankan ulang.

## Hindari

- Menaruh semua business logic di effect.
- Mengabaikan dependency array.
- Membuat effect yang saling memicu tanpa batas.
