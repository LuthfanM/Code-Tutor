# React Components

React app dibangun dari komponen kecil.

## Component Dasar

```tsx
function Welcome() {
  return <h1>Hello React</h1>
}
```

## Props

Props dipakai untuk mengirim data dari parent ke child component.

## Kebiasaan Baik

- Buat komponen fokus pada satu tanggung jawab.
- Pakai nama yang menjelaskan tujuan UI.
- Pindahkan logic berat ke helper atau hook.
