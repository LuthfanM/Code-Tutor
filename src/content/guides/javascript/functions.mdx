# JavaScript Functions

Function membantu memecah logic menjadi bagian kecil.

## Function Declaration

```js
function greet(name) {
  return `Hello, ${name}`
}
```

## Parameter

Parameter adalah input untuk function. Buat nama parameter jelas supaya code mudah dibaca.

## Return Value

Gunakan `return` saat function perlu menghasilkan value untuk dipakai lagi.
