# Deploy to Vercel

Vercel cocok untuk deploy Next.js karena integrasinya langsung.

## Checklist

- Pastikan `npm run build` berhasil.
- Isi environment variable di dashboard Vercel.
- Hubungkan repository GitHub.

## Environment

Untuk Supabase Auth, isi `NEXT_PUBLIC_SUPABASE_URL` dan `NEXT_PUBLIC_SUPABASE_ANON_KEY`.

## Domain

Setelah deploy stabil, arahkan domain `learn.luthfanm.com` lewat DNS.
