# Golang REST API with Gin

Gin adalah web framework ringan untuk membuat API di Go.

## Setup Router

```go
router := gin.Default()

router.GET("/health", func(c *gin.Context) {
  c.JSON(200, gin.H{"status": "ok"})
})

router.Run(":8080")
```

## Struktur Endpoint

Pisahkan handler, service, dan repository saat aplikasi mulai membesar.

## Praktik Awal

- Mulai dari endpoint kecil.
- Validasi input request.
- Kembalikan response JSON yang konsisten.
