# SQL Basic

SQL dipakai untuk membaca dan mengubah data di relational database.

## SELECT

```sql
select id, name, email
from users
where active = true;
```

## INSERT

Gunakan `insert` untuk menambah row baru.

## Safety

- Pakai `where` saat update dan delete.
- Pilih kolom yang dibutuhkan saja.
- Gunakan index untuk query yang sering dipakai.
