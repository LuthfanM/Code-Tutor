# Fullstack Next.js Path

Path ini disiapkan untuk fase setelah materi Next.js dan Supabase bertambah.

## Fokus

- Routing dan layout Next.js.
- Server component dan data fetching.
- Supabase Auth untuk login.
- Database untuk progress dan bookmarks.

## Target

Kamu bisa membangun aplikasi belajar dengan public content dan protected dashboard.
